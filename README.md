# ts-codegen

automate repeatdev process

## Usage
------
```bash
$ npm install -g ts-codegen
$ ts-codegen COMMAND
running command...
$ ts-codegen (-v|--version|version)
webpack-conf-cli/1.0.9 darwin-x64 node-v10.13.0
$ ts-codegen --help [COMMAND]
USAGE
  $ ts-codegen COMMAND
...
```

## Commands
--------
* ts-codegen new-lib
* ts-codegen remove-lib
* ts-codegen help [COMMAND]

## `ts-codegen new-lib`

create new lib

```bash
USAGE
  $ ts-codegen new-lib
```

## `ts-codegen remove-lib`

remove lib

```bash
USAGE
  $ ts-codegen remove-lib
```
